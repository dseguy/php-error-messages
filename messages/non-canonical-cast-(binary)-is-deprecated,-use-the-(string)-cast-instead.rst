Description
___________
 
The non-canonical cast ``(binary)`` has been deprecated. It should be replace by the simpler ``(string)``.

Example
_______

.. code-block:: php

   <?php
   
   $a = (binary) $string;
   
   ?>

Solutions
_________

+ Use ``(string)``.
