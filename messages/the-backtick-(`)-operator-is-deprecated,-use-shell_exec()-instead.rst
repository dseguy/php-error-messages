Description
___________
 
The back tick operator ``\`\``` is the equivalent to the function ``shell_exec()``. It is being deprecated since PHP 8.5.

Example
_______

.. code-block:: php

   <?php
   
   echo `ls`;
   
   ?>

Solutions
_________

+ Use shell_exec() instead.
